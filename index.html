Mission Brief

I'm a design engineer with 12+ years shipping production UI that bridges design intent and technical reality. I turn design systems from ideas into code, champion accessibility as a design constraint, and build tools that unlock creative work at scale.

I founded the UX Engineering discipline at Epsilon and grew our design system from a side project to company-wide adoption — writing components, documenting patterns, and embedding WCAG accessibility into the foundation. Now I'm building AI-powered creative tools that transform how designers and producers work: image-to-layers pipelines, natural language layout edits, and no-code platforms that generate thousands of ad variations from flexible templates.

Before the AI work, I built interactive installations with real-time rendering, gesture recognition, and motion design. The through-line has always been the same: I write the code that ships, and I care about how it feels.

Approach

  • Accessibility is a design constraint, not a checklist — I embed WCAG standards from day one, not as an afterthought. Accessible interfaces are better interfaces for everyone.
  • Production over prototypes — I write the code that ships. Designers call me when it needs to work, and engineers call me when the design intent isn't translating.
  • Systems thinking — I build reusable components, document design tokens, and create patterns that scale. One-off solutions create technical debt; systems create leverage.
  • Bridge the gap — I'm fluent in both design language and engineering constraints. I translate design intent into technical architecture and technical limitations into design opportunities.
